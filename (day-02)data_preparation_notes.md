Data Preparation for Machine Learning

 What is a Dataset?
A dataset is a collection of data used to train machine learning models.

It contains rows and columns.

Example:
- Rows = individual records
- Columns = features (information)

 Features vs Labels

 Features
Input data used for prediction.

Example:
- Area
- Bedrooms
- Location

Label
Output we want to predict.

Example:
- House Price

---

Types of Data

1. Numerical Data
Data in numbers:
- Age
- Salary
- Height

2. Categorical Data
Data in categories:
- Gender
- City
- Color

3. Text Data
Data in words:
- Reviews
- Emails
- Messages

Data Preprocessing
It means converting raw data into clean data for ML.

Steps:
- Cleaning data
- Handling missing values
- Encoding
- Feature selection

We divide data into:

- Train set → model learns here
- Test set → model is evaluated here

Common split:
- 80% Train
- 20% Test

 Why Good Data is Important?
Good data = better predictions
Bad data = wrong results
