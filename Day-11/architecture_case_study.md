# Deep Learning Architecture Case Study

| Problem                       | Type of Data   | Suitable Architecture | Reason                                                                                   |
| ----------------------------- | -------------- | --------------------- | ---------------------------------------------------------------------------------------- |
| Face Recognition              | Images         | CNN                   | CNN extracts visual features such as edges, textures, and facial patterns efficiently.   |
| Sentiment Analysis            | Text           | LSTM / Transformer    | These architectures understand the context and relationships between words.              |
| Machine Translation           | Text           | Transformer           | The Attention mechanism helps capture long-range dependencies between words.             |
| Speech Recognition            | Audio Sequence | LSTM / Transformer    | They effectively process sequential audio information and capture temporal dependencies. |
| Handwritten Digit Recognition | Images         | CNN                   | CNN is highly effective in identifying image patterns and handwritten digits.            |

---

# Summary

Choosing the correct deep learning architecture depends on the type of input data.

* Images → CNN
* Sequential Data → RNN
* Long Sequential Data → LSTM
* Large NLP Tasks and Long Context → Transformer

Using the appropriate architecture improves model accuracy, efficiency, and overall performance.
