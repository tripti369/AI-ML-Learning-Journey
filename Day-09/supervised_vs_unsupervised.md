# Supervised Learning vs Unsupervised Learning

| Aspect            | Supervised Learning                                    | Unsupervised Learning                         |
| ----------------- | ------------------------------------------------------ | --------------------------------------------- |
| Data Type         | Labeled Data                                           | Unlabeled Data                                |
| Objective         | Predict outcomes                                       | Discover hidden patterns                      |
| Target Variable   | Available                                              | Not Available                                 |
| Learning Style    | Learns from examples                                   | Learns from structure                         |
| Output            | Predictions                                            | Clusters/Patterns                             |
| Human Involvement | Higher                                                 | Lower                                         |
| Complexity        | Generally Simpler                                      | Often More Exploratory                        |
| Common Algorithms | Linear Regression, Logistic Regression, Decision Trees | K-Means, Hierarchical Clustering, PCA         |
| Use Cases         | Spam Detection, House Price Prediction                 | Customer Segmentation, Recommendation Systems |
| Evaluation        | Accuracy, Precision, Recall                            | Silhouette Score, Cluster Analysis            |

---

# Example

## Supervised Learning

### Input

| Hours Studied |
| ------------- |
| 2             |
| 4             |
| 8             |

### Output

| Result |
| ------ |
| Fail   |
| Pass   |
| Pass   |

The model learns the relationship and predicts future outcomes.

---

## Unsupervised Learning

### Input

| Age | Spending Score |
| --- | -------------- |
| 21  | 25             |
| 45  | 90             |
| 23  | 30             |

No labels are provided.

The algorithm automatically identifies groups of similar customers.

---

# Summary

Supervised Learning focuses on prediction using labeled data, while Unsupervised Learning focuses on discovering hidden structures and relationships in unlabeled data.
