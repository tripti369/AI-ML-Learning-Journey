 Features vs Labels

Features-
Features are input values used to make predictions.

Example:
- Area
- Bedrooms
- Location

 Labels
Labels are the output we want to predict.

Example:
- House Price

-House Price Prediction

Features:
- Area
- Bedrooms
- Location

Label:
- Price

---

Example 2: Student Marks Prediction

Features:
- Study Hours
- Attendance

Label:
- Marks

Difference Table

| Features | Labels |
|----------|--------|
| Input data | Output data |
| Independent | Dependent |
