# Case Study

## AI Coding Assistant

### Input

Natural language programming question.

Example:

"Write a Python program to sort a list."

---

### Expected Output

A correct Python solution with explanation.

---

### Why LLM?

- Understands programming languages
- Explains code
- Debugs errors
- Generates documentation
- Improves productivity

---

### Risks

- Incorrect code
- Hallucinated APIs
- Security vulnerabilities
- Outdated libraries

---

# Reflection

## Why are LLMs a breakthrough?

They can understand and generate human language across many tasks without training a separate model for each task.

---

## Why is tokenization important?

It converts human language into tokens that the model can process.

---

## Why do hallucinations occur?

Because the model predicts the most likely next token instead of verifying facts.

---

## Ethical Considerations

- Privacy
- Bias
- Misinformation
- Transparency
- Responsible AI
