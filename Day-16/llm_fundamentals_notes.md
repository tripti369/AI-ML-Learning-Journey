# Large Language Models (LLMs)

## What is an LLM?

A Large Language Model (LLM) is an AI model trained on massive amounts of text data to understand and generate human-like language.

Examples:
- GPT
- Llama
- Gemma
- Mistral

---

## LLM vs Traditional Machine Learning

Traditional ML:
- Uses structured datasets
- Solves specific tasks
- Requires feature engineering

LLM:
- Uses billions of text tokens
- Can perform many NLP tasks
- Learns language patterns automatically

---

## LLM vs Transformer

Transformer is the architecture.

LLM is a large model built using Transformer architecture.

Example:

Transformer → Architecture

GPT → Large Language Model

---

## Tokens

Tokens are small units of text processed by an LLM.

Example:

"ChatGPT is amazing"

↓

["Chat","GPT","is","amazing"]

---

## Tokenization

Tokenization is the process of converting text into tokens before sending it into the model.

---

## How LLM Generates Text

Input Prompt

↓

Tokenization

↓

Transformer

↓

Predict Next Token

↓

Generate Token

↓

Repeat until response is complete

---

## Training vs Inference

Training

- Model learns patterns
- Uses huge datasets
- Computationally expensive

Inference

- Uses trained model
- Generates responses
- Fast compared to training

---

## Context Window

Context Window is the maximum amount of text an LLM can remember during one conversation.

Larger context window = better understanding of long conversations.

---

## Hallucinations

Hallucinations occur when an LLM generates incorrect or fabricated information confidently.

Reasons:
- Missing knowledge
- Ambiguous prompts
- Prediction errors

---

## Limitations

- Hallucinations
- Bias
- Outdated knowledge
- High computational cost
- Privacy concerns
