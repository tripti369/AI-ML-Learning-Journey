# High Learning Rate vs Low Learning Rate

## High Learning Rate

A high learning rate means the model takes bigger steps while learning.

### Advantages

* Training becomes faster.
* The model reaches a solution in less time.
* Useful when we want quick results.

### Disadvantages

* The model can miss the best solution by taking very large steps.
* Training may become unstable.
* Sometimes the loss keeps increasing instead of decreasing.

---

## Low Learning Rate

A low learning rate means the model takes smaller steps while learning.

### Advantages

* Learning is more stable.
* The model has a better chance of finding the optimal solution.
* Usually gives better accuracy.

### Disadvantages

* Training takes more time.
* More epochs are required.
* Learning can become very slow.

---

## My Understanding

I learned that choosing the right learning rate is important because it directly affects how well the model learns. If the learning rate is too high, the model may learn incorrectly. If it is too low, the model learns very slowly. A balanced learning rate helps the model reduce errors efficiently and achieve better performance.
