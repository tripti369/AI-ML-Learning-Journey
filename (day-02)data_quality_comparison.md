Clean Data vs Unclean Data

Clean Data

- No missing values
- No duplicates
- Correct format
- Easy to analyze
- Better accuracy


Unclean Data
- Missing values
- Wrong entries
- Duplicate rows
- Noisy data
- Poor accuracy

Impact on ML Model

Clean Data:
- High accuracy
- Better predictions
- Faster training

Unclean Data:
- Wrong predictions
- Slow training
- Poor model performance

---

## Conclusion
Clean data is always required for good machine learning models.
