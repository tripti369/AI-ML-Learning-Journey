Neural Network Fundamentals

What is a Neural Network?
A neural network is a deep learning model inspired by the human brain that learns patterns from data.

Architecture
- Input Layer
- Hidden Layer
- Output Layer

Forward Propagation
Input → Weighted Sum + Bias → Activation Function → Output

Activation Functions

ReLU
Used in hidden layers.

Sigmoid
Used for binary classification.

Softmax
Used for multi-class classification.

Real-World Application
Face Recognition

Input: Face Image

Output: Person's Identity
